# Security Policy

This page explains how to report security issues for NUJ London Central Branch.

## Reporting
- Email: `security@nuj-lcb.org.uk`
- If you need secure upload or file transfer, use the `stfp.nuj-lcb.org.uk` service (file-server policy bucket). The `security.txt` file points to this page, and this page links to `https://stfp.nuj-lcb.org.uk/security-report` for any secure uploads you require.

## Response
We respond to confirmed issues within 72 hours and provide a public acknowledgement if appropriate.
